В этой папке хранятся только публичные JPG-копии документов.

Перед публикацией в них закрыты номера документов, регистрационные данные,
подписи, QR-коды и другие реквизиты, которые не нужны для подтверждения
образования и квалификации.

Оригиналы документов не следует добавлять в публичный репозиторий.
