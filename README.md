# Личный сайт врача-невролога

Одностраничный персональный сайт врача-невролога (к.м.н.), приём взрослых и детей.
Главный акцент — неврология, научная степень, опыт и места приёма. Кинезиология и
остеопатия показаны как дополнительные методы.

**Live:** https://maximusgenius-collab.github.io/Plusha/

## Структура

- `index.html` — вся страница (разделы: геро, о враче, с чем обращаются,
  методы, где принимаю, отзывы, FAQ, контакты) + SEO-метатеги и schema.org.
- `style.css` — стили и дизайн-токены.
- `img/` — изображения (фото врача, превью для соцсетей).

## Состояние данных

- Основные сведения о враче, научные профили, места работы и места приёма заполнены.
- Публичные копии документов обезличены: скрыты персональные номера, подписи и иные лишние реквизиты.
- Расписание, стоимость и запись намеренно вынесены на сайты клиник.
- Сайт не содержит форм и не собирает медицинские или контактные данные посетителей.
- Подготовлены Яндекс Метрика, управление согласием и политика конфиденциальности.
  Для запуска необходимо указать ID счётчика в метатеге `yandex-metrika-id`
  в `index.html` и `privacy.html`.
- Рейтинг и количество отзывов на ПроДокторов рекомендуется проверять раз в квартал.
- Раздел видео временно не публикуется; его можно вернуть после подготовки роликов и описаний.

Перед отдельным хостингом останется выбрать домен, систему аналитики и добавить
политику конфиденциальности под фактический набор счётчиков.
