# Личный сайт врача-невролога

Одностраничный персональный сайт врача-невролога (к.м.н.), приём взрослых и детей.
Главный акцент — неврология, научная степень, опыт и места приёма. Кинезиология и
остеопатия показаны как дополнительные методы.

**Live:** https://maximusgenius-collab.github.io/Plusha/

## Структура

- `index.html` — вся страница (разделы: геро, о враче, с чем обращаются,
  методы, где принимаю, отзывы, FAQ, контакты) + SEO-метатеги и schema.org.
- `style.css` — стили и дизайн-токены.
- `img/` — изображения (фото врача, превью для соцсетей).

## Что заменить на реальные данные

Места для правок помечены в `index.html` комментариями `ЗАМЕНИТЕ`:

- **ФИО врача** — в `<title>`, метатегах, schema.org, шапке, геро, подвале.
- **Образование, степень, стаж, кафедра** — раздел «О враче».
- **Клиники** — названия, адреса, метро, дни приёма, ссылки записи (раздел «Где принимаю»).
- **Отзывы** — реальные, с согласия пациентов.
- **Контакты** — телефон, мессенджеры, email.
- **Фото** — положите `img/doctor.jpg` (см. `img/README.txt`).
